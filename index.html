Acerca de Slick

Slick carousel es una librería de JavaScript que utiliza JQuery para facilitar la implementación de sliders dinámicos de imágenes

Con Slick Carousel puedes:

  • Crear sliders simples
  • Crear sliders con diferentes layouts
  • Elegir el número de imágenes a mostrar y scrollear
  • Optar entre scroll manual y automático
  • Añadir, eliminar y modificar las imágenes de la galería dinámicamente
